II. Написана программу на языке C#, которая:

В исходный файл Data1.xml в элемент Pay дописывает атрибут, который отражает сумму всех amount

Объяснение XSLT:

Объявление XML.
<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">: Определение XSLT таблицы стилей. <xsl:output method="xml" indent="yes"/>: Указывает, что выходной формат будет XML с отступами для читаемости. <xsl:template match="/">: Шаблон, который применяется к корневому элементу входного XML. <xsl:for-each select="Data/Employee">: Перебирает все элементы Employee внутри Data. : Создает новый элемент Employee в выходном XML. <xsl:value-of select="FullName"/>: Копирует значение элемента FullName из входного XML в элемент Name выходного XML. Аналогично для Position и Salary.
